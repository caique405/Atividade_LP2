# Atividade_LP2
Demonstração atividade sobre módulo Turtle
```json
  O módulo turtle é um módulo simples e divertido no qual se pode criar figuras a partir da programação!
  Pode ser uma ferramenta muito interessante para apresentar a programação para crianças pois sua 
  abordagem na construção das figuras são feitas atravéz de comandos muito semelhantes a comandos 
  de algoritmos!
  A ideia principal é uma seta ou um ponteiro que se comporta como se estivese em um 'Plano cartesiano' 
  e nele poderiamos atribuir comandos como para frente, para trás e assim por diante. 
  O módulo turtle é um módulo built in, ou seja, uma biblioteca nativda da linguagem pyhton. 
  Sua documentação oficial pode ser encontrada em:  https://docs.python.org/3.10/library/turtle.html
```
